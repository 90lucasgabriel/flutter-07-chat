  - Authentication with Google; 
- Anonymous messages are not allowed

## 6. Installing and Running
- Clone this repo:
```
git clone git@github.com:90lucasgabriel/flutter-07-chat.git chat
```

- Access directory:
```
cd chat
```

- Run
```
flutter run
```

- Check browser or connected device;
